== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `scroll-up-wordpress` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After complete activation process you will get a menu below settings named "Scroll To Top" in where change the option of this plugin.
4. Happy scrolling.....
